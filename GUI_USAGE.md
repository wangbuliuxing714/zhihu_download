# 知乎盐选工具 - 图形界面使用指南

## 快速开始

### 1. 启动程序

在项目目录下运行：

```bash
python run_gui.py
```

### 2. 界面介绍

图形界面包含以下几个主要区域：

#### 🔧 标题栏
- 显示程序名称和简介
- 采用现代化蓝色主题设计

#### ⚙️ 配置管理
- **Cookie输入框**：输入你的知乎Cookie
- **保存配置按钮**：点击保存Cookie到配置文件

#### 🎯 功能选择
- **爬取盐选单个问题**：开发中
- **爬取书的单个章节**：✅ 可用
- **爬取整本书**：开发中
- **关键词爬取**：计划中

#### 🔗 链接输入
- 输入要爬取的知乎市场链接
- 示例格式已在界面中显示

#### 📋 运行日志
- 实时显示程序运行状态
- 不同级别的日志用不同颜色区分：
  - 🟢 绿色：成功信息
  - 🔵 蓝色：普通信息
  - 🟡 黄色：警告信息
  - 🔴 红色：错误信息

#### 🎮 操作按钮
- **开始爬取**：开始执行爬取任务
- **清空日志**：清空日志窗口
- **关于**：查看程序信息

## 详细使用步骤

### 步骤1：获取Cookie

1. 打开浏览器，登录知乎（需要盐选会员账号）
2. 按 `F12` 打开开发者工具
3. 切换到 `Network`（网络）标签
4. 刷新页面
5. 点击任意一个请求
6. 在右侧找到 `Request Headers`（请求头）
7. 找到 `Cookie` 字段，复制整个Cookie值

### 步骤2：配置Cookie

1. 将复制的Cookie粘贴到GUI的Cookie输入框中
2. 点击"保存配置"按钮
3. 看到"配置保存成功"的提示即可

### 步骤3：选择功能

目前只有"爬取书的单个章节"功能可用，该选项默认已选中。

### 步骤4：输入链接

在URL输入框中输入知乎市场章节的链接，例如：

```
https://www.zhihu.com/market/paid_column/1702723501155422208/section/1788920608135983104
```

**如何获取链接？**
1. 在知乎APP或网页中找到你想爬取的盐选文章
2. 复制文章链接
3. 确保链接格式为 `https://www.zhihu.com/market/paid_column/...`

### 步骤5：开始爬取

1. 点击"开始爬取"按钮
2. 在日志窗口中查看实时进度
3. 等待爬取完成
4. 文件会保存在项目根目录，文件名为文章标题

## 常见问题

### Q: 提示"请先配置Cookie"？
A: 请按照步骤1和步骤2获取并配置Cookie。

### Q: 爬取失败，提示"内容提取失败"？
A: 可能的原因：
- Cookie已过期，需要重新获取
- URL格式不正确
- 没有盐选会员权限
- 网络连接问题

### Q: Cookie在哪里查看？
A: Cookie保存在项目根目录的 `config.ini` 文件中。

### Q: 如何更新Cookie？
A: 重新获取Cookie后，在GUI中输入并点击"保存配置"即可。

### Q: 程序卡住不动？
A: 爬取过程在后台线程运行，请耐心等待。可以在日志窗口查看进度。

### Q: 文件保存在哪里？
A: 文件保存在项目根目录，文件名为文章标题.txt。
