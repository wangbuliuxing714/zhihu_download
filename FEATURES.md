# 知乎盐选工具 - 功能特性

## 🎨 现代化图形界面

### 设计理念
- **扁平化设计**：采用现代扁平化设计风格，界面简洁美观
- **高对比度**：深色日志窗口配合浅色操作区，视觉层次分明
- **圆角设计**：所有按钮和卡片采用圆角设计，更加柔和友好
- **悬停反馈**：按钮悬停时颜色变化，提供即时视觉反馈

### 配色方案
- **主色调**：#4A90E2（专业蓝）
- **成功色**：#52C41A（翠绿）
- **警告色**：#FAAD14（橙黄）
- **错误色**：#F48771（珊瑚红）
- **紫色**：#722ED1（神秘紫）

## 🚀 技术特性

### 界面技术
- **框架**：Tkinter（Python标准库）
- **组件**：自定义ModernButton组件
- **布局**：Frame + Pack布局管理
- **字体**：Microsoft YaHei UI + Consolas

### 爬虫技术
- **请求库**：Requests
- **HTML解析**：lxml + etree
- **字体解析**：fonttools
- **OCR识别**：ddddocr
- **图像处理**：OpenCV + Pillow

### 性能优化
- **多线程**：爬取任务在后台线程运行
- **异步更新**：日志实时更新不阻塞
- **内存优化**：及时释放临时文件
- **错误恢复**：异常捕获和重试机制

## 💡 使用技巧

### 技巧1：快速配置
第一次使用时，建议先保存Cookie配置，之后就不需要每次输入了。

### 技巧2：复制日志
遇到错误时，可以选中日志内容按Ctrl+C复制，方便反馈问题。

### 技巧3：批量爬取
虽然GUI一次只能爬取一篇，但可以多次点击"开始爬取"，程序会依次处理。

### 技巧4：检查Cookie
如果爬取失败，首先检查Cookie是否过期，重新获取即可。

### 技巧5：保存链接
可以将常用的链接保存在文本文件中，需要时复制粘贴。
